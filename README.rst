Dependencies
------------

The following dependancies build the template on Ubuntu 12.04. Some of these
may not be required.

.. code-block:: console

    sudo apt-get install texlive texlive-science texlive-pstricks \
         texlive-pictures texlive-latex-extra texlive-extra-utils \
         dot2tex ps2eps
